Алгоритм решения

1. Вводим исходный массив.
2. Проверяем каждое значение введенного массива по очереди на соответствие условию задачи.
3. Если условия выполняются то данное значение записываем в новый массив.
4. Повторяем пункты 2 и 3 до тех пор пока не достигнем конца исходного массива.
5. Возращаем новый заполненый массив как результат.
